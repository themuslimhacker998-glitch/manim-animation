# 🎬 Manim Animations by Rashid

💻 Hi, I'm Rashid — I create **math and physics animations** using **Manim**.  
This repository contains my Manim animation projects as individual Python `.py` files.

---

## 🗂 Folder Structure

> Each folder contains a single `.py` file for that animation.

---

## 🚀 Projects Included

- **3d axis and cone** – transforming cone to cylinder
- **Integration** – visualizing integration math
- **differentiation** – teaching differential calculus in 10 minute
- **AM GM** – visualizing AM GM with geometery 
- **exponential** – making exponential graph and then solve the equation 
  

---

## 🛠 Tools Used

- Python 3.x  
- [Manim Community Edition](https://docs.manim.community/)  
- VS Code or Antigravity for editing `.py` files  

---

## 🎬 How to Run

1. Install Manim CE:  
```bash
pip install manim
Open terminal in the folder of a .py file
Run:
manim -pql filename.py SceneName

-pql → preview quality (fast rendering)
Replace SceneName with the name of the class inside the .py file
